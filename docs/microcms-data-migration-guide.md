# MicroCMS データ移行ガイド

## 概要

このドキュメントでは、残り3カテゴリ（watch, play, make）のデータをMicroCMSに移行する手順を説明します。

## 移行手順

### 1. MicroCMS管理画面にアクセス

1. https://hamayou-resort.microcms.io/ にログイン
2. 「contents-kokkofes」APIを選択
3. 「コンテンツを追加」ボタンをクリック

### 2. 各カテゴリのデータ入力

以下の各セクションに従って、カテゴリごとにデータを入力してください。

## トラブルシューティング

### 改行が表示されない場合

- descriptionフィールドの改行は、MicroCMSのテキストエリアでEnterキーを押して自然な改行として入力してください
- システムが自動的に `\n` → `<br>` 変換を行います

### 画像が表示されない場合

- MicroCMS管理画面で画像が正しくアップロードされているか確認
- galleryImagesフィールドに画像が追加されているか確認
- ブラウザのコンソールログでエラーメッセージを確認

### その他のエラー

- ブラウザのコンソールログを確認
- `npm run dev` のターミナル出力を確認
- データ形式が正しいか（categoryId, フィールド名など）を確認
